# Estrutura de Versionamento

O projeto utiliza o padrão:

```text
MAJOR.MINOR.PATCH
```

Exemplo:

```text
v0.5.0-beta
```

Onde:

| Campo | Significado              |
| ----- | ------------------------ |
| MAJOR | Alterações incompatíveis |
| MINOR | Novas funcionalidades    |
| PATCH | Correções e ajustes      |
